**📌 PS3C Robust Inference — Clinical Benchmark**

*With Prof. Balázs Harangi, Deputy Dean, University of Debrecen · Target: Medical Image Analysis / ISBI 2027*

| System | Test F1 | Eval F1 | Drop |
|--------|---------|---------|------|
| Best individual model (JNG) | 0.8702 | 0.8176 | -6.1% |
| Gradient Boost Stacking (baseline) | 0.9517 | 0.9245 | -2.7% |
| DPZ — worst collapse | 0.8622 | 0.7092 | **-17.7%** |
| Ours — TTA + Adaptive Ensemble + Conformal | 🔄 In Progress | 🔄 | — |

> Root cause: preprocessing-induced distribution shift between test and hidden evaluation sets — identified but never addressed in the original paper. Our three-stage framework applies architecture-aware TTA (LayerNorm for ViTs, TENT for CNNs), sample-adaptive ensemble weighting, and split-conformal selective prediction with 95% coverage guarantees.
